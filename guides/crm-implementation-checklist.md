# CRM Implementation Checklist

A step-by-step checklist for planning, executing, and validating a CRM implementation
in a Marketing Operations context.

---

## 📋 Phase 1 — Discovery & Planning

- [ ] Define business objectives for the CRM
- [ ] Identify key stakeholders (Marketing, Sales, RevOps, IT)
- [ ] Document current lead and contact data sources
- [ ] Map existing processes (lead capture, handoff, follow-up)
- [ ] Define success metrics (adoption rate, data quality score, pipeline visibility)
- [ ] Choose CRM platform (HubSpot, Salesforce, etc.)
- [ ] Set project timeline and assign ownership

---

## 🗂 Phase 2 — Data Preparation

- [ ] Audit existing data (contacts, companies, deals)
- [ ] Define data fields and custom properties needed
- [ ] Establish data hygiene rules (duplicates, formatting, required fields)
- [ ] Map old data fields to new CRM fields
- [ ] Clean and deduplicate data before migration
- [ ] Define segmentation criteria (lifecycle stages, lead scores, regions)

---

## ⚙️ Phase 3 — CRM Configuration

- [ ] Set up users, roles, and permissions
- [ ] Configure pipeline stages aligned with sales process
- [ ] Build lead capture forms and connect to CRM
- [ ] Set up lifecycle stage definitions and automation rules
- [ ] Configure lead scoring model
- [ ] Build email templates and sequences
- [ ] Integrate with key tools (marketing automation, ad platforms, support)
- [ ] Set up dashboards and reporting views

---

## 🔁 Phase 4 — Testing

- [ ] Test lead capture forms end-to-end
- [ ] Validate automation triggers and workflows
- [ ] Test data sync across integrated tools
- [ ] Run UAT (User Acceptance Testing) with key stakeholders
- [ ] Verify reporting accuracy against source data
- [ ] Document and resolve all bugs before go-live

---

## 🚀 Phase 5 — Go-Live & Adoption

- [ ] Migrate final cleaned data into CRM
- [ ] Conduct team training sessions (Marketing, Sales, Leadership)
- [ ] Create internal user documentation / SOPs
- [ ] Set a 30-day post-launch review checkpoint
- [ ] Monitor adoption metrics (login rate, data entry quality)
- [ ] Establish an ongoing data governance process

---

## 📝 Notes

> Customize this checklist based on your CRM platform and company size.
> Built from hands-on experience in Marketing Operations.
